Navegación SPA: Las aplicaciones de una sola página, de ahora en más SPA, son aplicaciones web que cargar una sola página html y actualizan dinámicamente el contenido a medida que el usuario interactua con la aplicación

¿Cual es el problema de la navegación SPA?

El problema principal en las SPAs es la gestión del historial en el navegador y la navegación entre páginas sin recargar la página completa
